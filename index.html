Our Solution

Nature's Vigil is an edge-based computer vision system that detects littering behaviors in real-time CCTV footage and triggers automated enforcement alerts (audio warnings, visual overlays, incident logging, and municipal notification).

Our system works through a six-step process that ensures accurate detection and immediate response:

  1. Data Ingestion: Standard IP cameras capture continuous video streams
  2. Real-time Processing: YOLO v8 detects humans and waste objects with 30-60 FPS performance
  3. Behavior Recognition: MediaPipe extracts 33 body landmarks per frame, analyzing arm trajectory and temporal patterns
  4. Action Classification: Temporal CNN classifies sequences as littering vs. normal activity with 150ms latency
  5. Alert Trigger: Upon detection (90% confidence), system activates audio alarm, visual overlay, SMS/app notification to enforcement officers
  6. Evidence Preservation: 24/7 encrypted logging enables accountability and supports enforcement proceedings

Technology Stack

Our system leverages cutting-edge computer vision and AI technologies to detect littering behavior in real-time:

Architecture: Two-Stage Approach

Stage 1: Object Detection (YOLO v8)
Detects humans and waste objects in real-time with 30-60 FPS performance. Multiple model variants (nano to extra) provide speed/accuracy tradeoff for different deployment scenarios.

Stage 2: Action Recognition
Uses MediaPipe for pose estimation (33 body landmarks per frame), analyzing arm trajectory and angle changes to identify throwing actions. Temporal CNN captures sequential movement patterns for accurate behavior classification.

Recommended Tech Stack

  • Language: Python 3.9+
  • Deep Learning: PyTorch (over TensorFlow for beginners)
  • Models: YOLOv8, MediaPipe Pose, Custom Temporal CNN
  • Deployment: NVIDIA Jetson Xavier NX for edge processing
  • Tools: OpenCV, Label Studio/Roboflow for annotation
